# M-R Ecommerce

متجر إلكتروني عربي حديث مبني بـ Next.js وReact وTailwind CSS.

## التشغيل

```bash
npm install
npm run dev
```

انسخ `.env.example` إلى `.env.local` وعدل بيانات قاعدة البيانات وحساب الإدارة قبل التشغيل الإنتاجي.

## الصفحات

- `/` الصفحة الرئيسية
- `/products` المنتجات مع البحث والفلترة والترتيب
- `/products/[slug]` تفاصيل المنتج
- `/checkout/[slug]` نموذج الطلب المختصر
- `/order-success` صفحة نجاح الطلب
- `/admin` لوحة الإدارة

## قاعدة البيانات

تمت إضافة مخطط Prisma في `prisma/schema.prisma` للمنتجات، الطلبات، العملاء، التصنيفات، الإعدادات، ومستخدمي الإدارة.

## ملاحظات إنتاجية

النسخة الحالية تستخدم بيانات عرض داخل `lib/data.ts`. عند الانتقال للإنتاج اربط Prisma بقاعدة PostgreSQL، واستبدل تسجيل الدخول التجريبي بجلسات موقعة وكلمات مرور مشفرة عبر `bcrypt` أو `argon2`.
